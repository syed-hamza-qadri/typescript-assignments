# Typescript-Assignments

# Typescript
<!-- Install Typescript -->
npm install --global typescript@5.0.2 
<!-- Check ts version -->
tsc --version / --v

# Commands
<!-- Compile ts to js -->
tsc main.ts
<!-- Run Js file -->
node main.js
<!-- Create tsconfig.json file -->
tsc --init
<!-- Create package.json file -->
npm init -y
<!-- Create 2 Files node-modules & package-lock.json -->
npm i @types/node -D
